Объяснение кода
EEPROMHandler.h
Метод begin:

Инициализирует EEPROM, указывая размер EEPROM в байтах. Этот метод должен вызываться в setup().
Метод saveFloat:

Принимает адрес в EEPROM и значение типа float, которое нужно сохранить.
Разделяет float значение на байты и сохраняет их в EEPROM по указанному адресу.
Использует EEPROM.commit(), чтобы записать изменения в память.
Метод readFloat:

Принимает адрес в EEPROM и считывает значение типа float с этого адреса.
Собирает байты из EEPROM и преобразует их обратно в значение типа float.
main.cpp
Инициализация EEPROMHandler:

Создается объект EEPROMHandler.
Метод setup:

Инициализирует последовательный порт и EEPROM.
Пример сохранения значения 123.456 в EEPROM по адресу 0 и вывод этого значения в Serial Monitor.
Пример чтения значения из EEPROM по адресу 0 и вывод его в Serial Monitor.
Этот код демонстрирует, как сохранить и прочитать значение типа float в EEPROM на ESP8266, используя класс EEPROMHandler.
